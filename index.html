¿Que es FEDORA?

Fedora es una opción ideal para implementar servidores web y bases de datos. Puedes configurar y administrar servidores web populares como Apache, Nginx o Lighttpd. Además, Fedora admite bases de datos robustas como PostgreSQL, MySQL y MariaDB, proporcionando una base sólida para aplicaciones web y servicios empresariales.

Descripcion

Fedora utiliza una licencia de código abierto que promueve la libertad de uso, distribución y modificación del software.

Algunos componentes y paquetes en Fedora están bajo la licencia MIT, que permite un mayor grado de libertad en el uso y distribución.

Variantes

Además de Fedora Server, existen otras variantes de Fedora, como Fedora Workstation (orientada a estaciones de trabajo), Fedora Cloud (diseñada para implementaciones en la nube), y Fedora Silverblue (que utiliza tecnología de contenedores para la administración del sistema).

Servicios que proporciona

Servidores WEB

Apache: Fedora es compatible con el servidor web Apache, uno de los servidores web más populares y ampliamente utilizados en el mundo. Apache es conocido por su flexibilidad y robustez, y es capaz de alojar una variedad de sitios web, desde simples páginas estáticas hasta aplicaciones web dinámicas.

Servidores de base de datos

MySQL y MariaDB: Fedora es compatible tanto con MySQL como con MariaDB, que es un fork de MySQL. Ambos son sistemas de gestión de bases de datos relacionales ampliamente utilizados en aplicaciones web y empresariales.

Almnacenamiento en red

NFS (Network File System): Fedora Server es adecuado para implementar soluciones de almacenamiento en red utilizando el protocolo NFS. NFS permite que los sistemas operativos compartan archivos y directorios a través de la red, facilitando el acceso y el intercambio de datos entre sistemas.

Administración de usuarios y autenticación

LDAP (Lightweight Directory Access Protocol):Fedora admite servicios de autenticación mediante LDAP, que es un protocolo de acceso a directorios utilizado para gestionar y organizar información de usuarios en un sistema distribuido.

Gestión de Paquetes

Fedora utiliza el sistema de gestión de paquetes DNF (Dandified Yum), que facilita la instalación, actualización y eliminación de software.

Seguridad

Fail2Ban: Una herramienta de seguridad para proteger servidores contra ataques de fuerza bruta.

Mision

Fedora, como sistema operativo de servidor, tiene como misión fundamental la promoción del software libre y de código abierto, asegurando la libertad de los usuarios para ver, modificar y distribuir el software. La distribución se destaca por su enfoque en la innovación y la adopción de tecnologías emergentes, sirviendo como un terreno de pruebas para las últimas novedades en sistemas operativos. La seguridad es una prioridad, incorporando características y prácticas recomendadas para garantizar un entorno seguro y confiable.
